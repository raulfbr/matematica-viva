## O Ciclo CPA (Concreto → Pictórico → Abstrato)

```
     ENACTIVE           ICONIC            SYMBOLIC
    (Concreto)    →   (Pictórico)    →   (Abstrato)
        │                  │                  │
   Mãos na massa      Olhos que veem    Mente que pensa
        │                  │                  │
   Objetos físicos    Desenhos/Modelos   Números/Símbolos
```

---

## Checklist

### 1. Fase ENACTIVE (Concreto)

| Critério | ✅/❌ | Observação |
|----------|-------|------------|
| Objeto físico identificado? | | |
| Criança manipula com as mãos? | | |
| Material acessível (casa comum)? | | |
| Instruções claras para o Portador? | | |
| Tempo: 5-7 minutos? | | |

**Exemplos válidos:**
- Feijões, botões, pedras
- Blocos de construção
- Frutas, legumes
- Palitos de sorvete
- Moedas reais

**Exemplos INVÁLIDOS:**
- "Imagine que você tem 5 maçãs" (não é concreto!)
- Flashcards (já é pictórico)
- Vídeos (passivo, não manipulativo)

---

### 2. Fase ICONIC (Pictórico)

| Critério | ✅/❌ | Observação |
|----------|-------|------------|
| Representação visual presente? | | |
| Conecta ao objeto concreto? | | |
| Criança cria/desenha? (ideal) | | |
| Ou observa modelo? (aceitável) | | |
| Tempo: 3-5 minutos? | | |

**Exemplos válidos:**
- Desenhar os 5 feijões no papel
- Number Bonds desenhados
- Bar Models
- Diagrama de conjuntos
- Ilustração do livro

**Exemplos INVÁLIDOS:**
- Pular direto para "5 + 3 = 8"
- Mostrar apenas o símbolo
- Não ter transição visual

---

### 3. Fase SYMBOLIC (Abstrato)

| Critério | ✅/❌ | Observação |
|----------|-------|------------|
| Notação matemática introduzida? | | |
| Conecta ao pictórico? | | |
| Criança escreve/diz o símbolo? | | |
| Não é imposto, mas descoberto? | | |
| Tempo: 3-5 minutos? | | |

**Exemplos válidos:**
- "Isso que você desenhou, os matemáticos escrevem assim: 5"
- "Os 3 grupos que você fez, olha só: 3 × 4 = 12"
- Apontar para o desenho e depois para o numeral

**Exemplos INVÁLIDOS:**
- Começar pela equação
- "Decore: 2 + 2 = 4"
- Símbolo sem conexão com concreto/pictórico

---

## Ordem Importa!

```
❌ ERRADO: Simbólico → Pictórico → Concreto (inverso)
❌ ERRADO: Concreto → Simbólico (pula pictórico)
❌ ERRADO: Pictórico → Simbólico (pula concreto)

✅ CERTO: Concreto → Pictórico → Simbólico (sempre nesta ordem)
```

---

## Pergunta Final

> *"Se eu tapar a seção simbólica, a criança ainda entendeu o conceito usando apenas o concreto e o pictórico?"*

Se a resposta for **SIM**, o CPA está bem feito.
Se a resposta for **NÃO**, o simbólico está carregando peso demais.
